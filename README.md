# RodaCash - Projeto Fintech

Tela única desenvolvida para a atividade avaliativa de HTML, CSS e Tailwind CSS.

## Arquivos principais

- `index.html`: página principal do projeto.
- `css/estilos.css`: estilos personalizados separados do HTML.
- `documentos/LINK_REPOSITORIO.pdf`: PDF com o link público do repositório GitHub.

## Link do repositório

https://github.com/Guialmeida3544/Rodacash_Fintech

## Como abrir

Abra o arquivo `index.html` no navegador.

## Observação

O projeto utiliza Tailwind CSS por CDN e CSS personalizado em arquivo separado. Não há JavaScript próprio no projeto.
